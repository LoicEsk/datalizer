=== Datalizer ===
Contributors: Loïc Laurent
Requires at least: 4.1.0
Tested up to: 4.2.4

WordPress plugin to save and analiyse datas from physical sensors

== Installation ==
This plugin is hosted on github. You can use the GitHub Updater to install and update.
